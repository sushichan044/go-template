# go-template

## Setup

### Scaffold

1. `tiged sushichan044/go-template`
2. Replace `module github.com/sushichan044/go-template` in `go.mod` with our own module path.
3. Set `formatters.settings.goimports.local-prefixes` to our own module path (for example, github.com/acme/service).

### Agent Skills

Add needed skills via `bunx skills add samber/cc-skills-golang`.
